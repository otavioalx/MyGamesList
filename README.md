# MyGamesList
> Uma API moderna para gamers que desejam organizar sua biblioteca, acompanhar o progresso e gerenciar seu backlog de jogos de forma eficiente.
---
## Sobre o Projeto
O **MyGamesList** foi desenvolvido para ser o companheiro ideal de qualquer jogador. Através de uma interface backend robusta, o sistema permite o gerenciamento completo de coleções, desde títulos clássicos até os lançamentos mais recentes, garantindo que você nunca perca o controle do que está jogando.
